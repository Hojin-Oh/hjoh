

한글이 깨지지 않게 하려면 **저장 방식**이 가장 중요합니다.

1. PC에서 **메모장(Notepad)**을 엽니다.
    
2. 위의 코드 중 필요한 박스 내용을 복사해서 붙여넣습니다.
    
3. **[파일]** -> **[다른 이름으로 저장]**을 클릭합니다.
    
4. 저장 설정 (★중요):
    
    - **파일 이름:** `신규프로젝트생성.bat` (뒤에 `.bat`를 꼭 직접 적어야 합니다.)
        
    - **파일 형식:** `모든 파일 (*.*)` 선택
        
    - **인코딩(Encoding):** 반드시 **`ANSI`** (윈도우 10/11 기본)로 저장합니다.
        
    - _(참고: 코드 상단의 `chcp 65001` 명령어가 UTF-8 한글 출력을 제어하지만, 배치 파일 자체는 ANSI로 저장해야 CMD 창에서 오류 없이 잘 작동하는 경우가 많습니다. 만약 한글이 깨진다면 인코딩을 `UTF-8`로 바꿔서 다시 저장해보세요.)_
        
5. 저장된 파일을 더블클릭하면 검은 창이 뜨면서 실행됩니다.
    

**만드시는 데 어려움이 있거나, 추가하고 싶은 기능(예: 엑셀 파일 자동 생성 등)이 있다면 말씀해 주세요.**
